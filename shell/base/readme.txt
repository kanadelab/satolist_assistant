
さとりすとのアシスタントとはじめてのゴーストのためのキット
シェルについて


2018.9.25 ななっち

■このシェルについて
このシェルは、「さとりすとのアシスタント」ゴーストのシェルですが、
ゴーストを作ってみたいという人に使ってもらうためのシェルでもあります。

このシェルは一般的なフリーシェルと同じように扱っていただけます。
ゴーストを作るためにそのまま使用したり、改変して使ったり、作ったゴーストを作品として公開してもかまいません。

「さとりすとのアシスタント」として使われているパーツ以外にも
目つきや胸の大きさ、獣耳やめがねなどアクセサリなどがいくつかありますので
すきな組み合わせで使ってみてくださいね。

ただしここに同梱していませんがアシスタントで使用している「さとりすとのアイコンの髪飾り」だけは、
「さとりすとのアシスタント」の目印にしておきたいので、公開する場合は使用されないようにお願いします。


■使用可能範囲について
無用なトラブルを避けるために、本体画像の利用はゴーストのシェルとしての利用のみでお願いします。
本体画像を使わずシェルをもとにしたイラストなどデザインの利用はこの他でも行ってかまいません。
このシェルは成人向けゴーストに使用していただいてかまいません。


■シェル内容の説明
shell.txt に簡単なものですがまとめています。
